## ▶️ Como Executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/html-lab-simulator.git

# Acesse a pasta
cd html-lab-simulator

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```
